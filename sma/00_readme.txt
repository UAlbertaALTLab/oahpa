This directory contains the files relevant to the smaoahpa application.
 - src: the source files with the lexicon smaX (i.e., smanob, smaswe)
 - nobsma: the reverted files from smanob to nobsma
 - swesma: the reverted files from smaswe to swesma
Caveat: the reverted files are already frozen, i.e., they are ready for extension for synonyms and the like.
