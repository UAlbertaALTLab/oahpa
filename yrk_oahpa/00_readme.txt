Directory for the yrk_oahpa with new code (like sjd_oahpa, sma_oahpa and univ_oahpa).

Bugs in Leksa:
мяˮ => tent
юр” => butter
иˮ => water
сэрˮ => salt
саˮла => stupid
выʼ => tundra
ӈодяʼ иˮ => juice
 ==> check data and pipeline!

Note:
-------- letters:
ˮ = non-nasalising glottal stop = U+02EE
ʼ = nasalising glottal stop = U+02BC

-------- punctuation marks:
” = right double quotation mark = U+201D
’ = right single quot mark = U+2019

-------- xml- and punctiation marks:
" = quotation mark = U+0022
' = apostrophe = U+0027

Nenets uses 02EE and 02BC as letters.
So these are not bugs.
